# Cyber Attack Detection System

## Project Description

The **Cyber Attack Detection System** is a web-based application built using **Python and Flask** that helps identify potential cyber attacks by analyzing network traffic data.
The system takes inputs such as **Duration, Source Bytes, and Destination Bytes** and checks whether the activity is normal or suspicious.

This project demonstrates the use of **Python, Flask, and basic machine learning logic** to detect possible cyber threats in a simple and interactive web interface.

---

## Features

* Detects possible cyber attacks based on network data
* Simple and user-friendly web interface
* Input fields for:

  * Duration
  * Source Bytes
  * Destination Bytes
* Instant result prediction
* Built using Python and Flask
* Lightweight and easy to run locally

## How to Run the Project

### 1. Clone the repository

git clone https://github.com/shrikantkumarofficalin-cyber/cyber-attack-detection-system.git

### 2. Go to project folder

cd cyber-attack-detection-system

### 3. Create virtual environment

python -m venv .venv

### 4. Activate environment

.venv\Scripts\activate

### 5. Install dependencies

pip install -r requirements.txt

### 6. Run the application

python app.py

### 7. Open browser

http://127.0.0.1:5000
